Projet-DevOps-ImaneBassou

Ce mini-projet vise à mettre en œuvre les principes fondamentaux du DevOps à travers la création d’une chaîne d’intégration continue (CI). Il utilise un dépôt GitHub, des workflows GitHub Actions et un pipeline Jenkins pour automatiser le cycle de vie d’une application Java simple, servant de support pédagogique pour illustrer l’automatisation du build, l’exécution des tests, la génération d’artefacts et la notification des résultats du pipeline. Le projet permet de comprendre les concepts de CI, de structurer un dépôt Git, d’automatiser les builds et tests avec GitHub Actions, de configurer un pipeline Jenkins pour compiler et archiver les artefacts, d’assurer la traçabilité via les commits et Pull Requests, et de notifier l’état du pipeline via Slack. L’architecture repose sur GitHub pour le code source, Java/Maven pour l’application, GitHub Actions pour la CI, Jenkins pour l’orchestration, et Slack pour les notifications. Les technologies utilisées incluent Git, GitHub Actions, Jenkins, Java 17, Maven, Docker et Slack. La structure du projet comprend le dossier hello-devops avec le code source et le pom.xml, le dossier .github/workflows avec le workflow CI et le fichier README.md. Le projet est géré avec deux branches principales (main pour la version stable et dev pour le développement), et le workflow CI compile automatiquement le code, exécute les tests unitaires et génère le package JAR, tandis que Jenkins centralise les builds et archive les artefacts. Grâce à cette mise en place, la CI fonctionne pleinement, les builds et tests s’exécutent automatiquement, les artefacts sont archivés et les notifications Slack sont envoyées. Des améliorations futures peuvent inclure le déploiement continu (CD), Docker Compose, des tests avancés, la sécurisation des accès Jenkins et GitHub, ainsi que l’ajout de métriques et monitoring. Ce projet démontre concrètement l’importance de l’automatisation dans le cycle de développement logiciel et constitue une base solide pour des projets DevOps plus avancés.

Auteur : Imane Bassou
